GitHub

Logado em sua conta do GitHub:

  • Acesse https://github.com/settings/tokens.
  • Clique em [Generate new token]
  • Informe um nome para o token, sugestão: divulgit
  • Informe uma data de expiração
  • Selecione a permissão: [repo - Full control of private repositories]
  • Clique [Generate token]


    • GitLab

    Logado em sua conta do GitLab:

  • Acesse https://gitlab.com/-/profile/personal_access_tokens.
  • Informe um nome para o token, sugestão: divulgit
  • Informe uma data de expiração
  • Selecione as permissões: read_api
  • Clique [Create personal access token]
