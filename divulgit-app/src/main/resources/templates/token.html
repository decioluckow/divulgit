Azure

Logado em sua conta do Azure DevOps:

  • Clique no icone "User Settings" na barra superior, e acione "Personal Access Tokens"
  • Clique em [New Token]
  • Informe um nome para o token, sugestão: divulgit
  • Informe para Organization o valor All accessible organizations
  • Informe uma data de expiração
  • Para Scopes, selecione : [Custom defined]
  • Nos escopos, marque: Code > Read
  • Clique [Create]


    • GitHub

    Logado em sua conta do GitHub:

  • Acesse https://github.com/settings/tokens.
  • Clique em [Generate new token]
  • Informe um nome para o token, sugestão: divulgit
  • Informe uma data de expiração
  • Selecione a permissão: [repo - Full control of private repositories]
  • Clique [Generate token]


    • GitLab

    Logado em sua conta do GitLab:

  • Acesse https://gitlab.com/-/profile/personal_access_tokens.
  • Informe um nome para o token, sugestão: divulgit
  • Informe uma data de expiração
  • Selecione as permissões: read_api
  • Clique [Create personal access token]


    • BitBucket

    Logado em sua conta do BitBucket:

  • Acesse https://bitbucket.org/account/settings/app-passwords.
  • Clique no botão para criar um "app password"
  • Informe um nome para o token, sugestão: divulgit
  • Selecione as permissões abaixo e clique em "Create" para gerar o token
    • Account: Email e Read
    • Repositories: Read
    • Pull requests: Read
